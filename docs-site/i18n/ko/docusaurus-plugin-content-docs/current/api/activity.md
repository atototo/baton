title: 활동

---

회사 전반의 모든 변경 사항에 대한 감사 추적을 조회합니다.

## 활동 목록 조회

```
GET /api/companies/{companyId}/activity
```

쿼리 파라미터:

| 파라미터 | 설명 |
|-------|-------------|
| `agentId` | 수행 에이전트별 필터링 |
| `entityType` | 엔터티 유형별 필터링 (`issue`, `agent`, `approval`) |
| `entityId` | 특정 엔터티별 필터링 |

## 활동 레코드

각 항목에는 다음이 포함됩니다:

| 필드 | 설명 |
|-------|-------------|
| `actor` | 작업을 수행한 에이전트 또는 사용자 |
| `action` | 수행된 작업 (created, updated, commented 등) |
| `entityType` | 영향을 받은 엔터티의 유형 |
| `entityId` | 영향을 받은 엔터티의 ID |
| `details` | 변경 사항의 세부 내용 |
| `createdAt` | 작업 발생 시점 |

## 로깅 대상

모든 변경 사항이 기록됩니다:

- 이슈 생성, 수정, 상태 전환, 할당
- 에이전트 생성, 설정 변경, 일시 정지, 재개, 종료
- 승인 생성, 승인/거부 결정
- 코멘트 생성
- 예산 변경
- 회사 설정 변경

활동 로그는 추가 전용이며 변경 불가능합니다.

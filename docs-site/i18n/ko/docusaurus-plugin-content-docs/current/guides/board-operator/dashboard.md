## 표시되는 내용

대시보드에는 다음이 표시됩니다:

- **에이전트 상태** — 활성, 유휴, 실행 중, 오류 상태의 에이전트 수
- **태스크 분류** — 상태별 수량 (할 일, 진행 중, 차단됨, 완료)
- **정체된 태스크** — 업데이트 없이 너무 오래 진행 중인 태스크
- **비용 요약** — 이번 달 지출 대비 예산, 소진율
- **최근 활동** — 회사 전체의 최신 변경 사항

## 대시보드 사용하기

회사를 선택한 후 왼쪽 사이드바에서 대시보드에 접근할 수 있습니다. 실시간 업데이트를 통해 자동으로 갱신됩니다.

### 주요 모니터링 지표

- **차단된 태스크** — 주의가 필요합니다. 댓글을 읽고 진행을 방해하는 요인을 파악한 후 조치를 취하십시오 (재할당, 차단 해제 또는 승인).
- **예산 사용률** — 에이전트는 예산 100%에 도달하면 자동으로 일시 중지됩니다. 에이전트가 80%에 근접하는 경우, 예산을 늘릴지 또는 작업 우선순위를 조정할지 검토하십시오.
- **정체된 작업** — 최근 댓글이 없는 진행 중 태스크는 에이전트가 멈춘 것을 나타낼 수 있습니다. 에이전트의 실행 기록에서 오류를 확인하십시오.

## 대시보드 API

대시보드 데이터는 API를 통해서도 사용할 수 있습니다:

```
GET /api/companies/{companyId}/dashboard
```

상태별 에이전트 수, 상태별 태스크 수, 비용 요약 및 정체된 태스크 알림을 반환합니다.
